Q. No1. What is DOM and what is the purpose of DOM?

Ans:- DOM is a programming API for Document Object Model in JavaScript. A web page is a document that can be displayed in a browser window or as HTML source.
In both cases, it is the same document but the Document Object Model (DOM) representation allows it to be manipulated.
As an object-oriented representation of the web page, it can be modified with scripting languages such as JavaScript.

Q. No2. How will you select HTML elements using DOM? Name the DOM methods.

Ans:- I select HTML elements using DOM in 6 ways:-
1. getElementById () - search element by element_id
2. getElementsByTagName () - search element by tag name
3. getElementsByClassName () - search element by class name
4. getElementsByName () - search element by name attribute
5. querySelector () - returns the first element that matches the specified selector
6. querySelectorAll () - returns elements that match the specified selector

Q. No3. What is event bubble?

Event bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element, and both elements have registered a handle to that event. It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy.

Q. No4. What is a callback function and why will you use it?

A JavaScript callback is a function which is to be executed after another function has finished execution. A more formal definition would be - Any function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function
